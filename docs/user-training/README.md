## Learning Paths

### Business Owner
- Focus: Dashboard, Invoices, Reports
- Time: 1.5 hours
- Start: Getting Started → Dashboard → Invoices → Reports

### Accountant
- Focus: All features, Advanced reporting
- Time: 3 hours
- Start: Complete all training guides

### Team Member
- Focus: Basic features, Collaboration
- Time: 1 hour
- Start: Getting Started → Dashboard → Expenses
